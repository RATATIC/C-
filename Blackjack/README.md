	Описание классов 	

	Card - Карта для игры 

	Hand - набор карт 

	Deck - произвобный класс от Hand ,который собой представляет колоду 

	Person - абстрактный класс , который обобщенно описывает собой игрока

	Player - игрок 

	Dealer - дилер

	Game - процесс игры
